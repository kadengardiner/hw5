PROJECT DESCRIPTION
----------------------------------------------------------------------------
This is a one-line Scrabble game built with HTML, CSS, and JavaScript.
It uses jQuery UI for drag-and-drop functionality. Players can drag letter
tiles from their rack onto the board to form words and score points.

HOW TO RUN
----------
Visit: https://kadengardiner.github.io/hw5/

IMPLEMENTED FEATURES
-----------------------------------------------------------------------------
All features are fully working:

- Random tile distribution from proper Scrabble tile bag (100 tiles total)
- Drag and drop tiles from rack to board
- Tiles snap to board squares and can be returned to rack
- One-line board with 3 bonus square types: Double Letter, Double Word, Triple Letter
- Real-time score calculation with bonus multipliers applied correctly
- Submit word button to finalize score and deal new tiles
- Clear board button to return tiles without scoring
- New tiles button to refill hand
- Restart game button to reset everything
- Validation that tiles must be placed consecutively (no gaps)
- Tiles can only be dropped on board or rack (bounce back elsewhere)
- Score tracking across multiple words
- Tiles remaining counter shows how many tiles are left in the bag
